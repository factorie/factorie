ChainNER2

object ChainNER2

Simple, introductory linear-chain CRF for named-entity recognition, using FACTORIE's low-level "imperative" language to define model structure.

Demonstrates model creation, training and testing. Overly simple features to not, however, provide very high accuracy. See ChainNER3 for a related example with better features.
